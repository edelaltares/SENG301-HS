#How to build

1. Download files to computer
2. Navigate to dist folder
3. Open a terminal
4. Type in "java -jar Project.jar"

Alternatively,

1. Download files to computer
2. Download NetBeans
3. Open project in NetBeans
4. Navigate to Main class (in GUI package)
5. Run Main class
